# AI Security Testing — HR Assistant

## Project Overview
A security testing project that evaluates an AI HR assistant against prompt-based attacks.

## Attack Categories Tested
- Prompt Injection
- Jailbreak Attempts
- System Prompt Extraction
- Data Leakage
- Tool Misuse Simulation
- Multi-turn Manipulation

## Tools Used
- OpenRouter API (free model)
- Supabase (database)
- Python

## Results
- Total Tests: 15
- Pass: 13
- Fail: 2
